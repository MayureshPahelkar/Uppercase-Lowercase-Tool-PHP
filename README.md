# Uppercase-Lowercase-Tool-PHP

The Text Case Converter is a simple and interactive web application built using HTML, CSS, and PHP. It allows users to enter any text into an input field and instantly convert it into uppercase or lowercase format using dedicated buttons. The converted result is displayed on the same page below the buttons without reloading or redirecting.

This project demonstrates basic form handling and string manipulation in PHP, making it ideal for beginners learning backend development.
